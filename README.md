RESP-SigmaHole: Restrained Electrostatic Potential Charge for Sigma Hole
==================================

## Requirement
- Psi4
- RDKit
- Acpype
- AmberToools

## Installation & Usage
1. Install required package
2. run RESP.sh to run examples
3. read and modify RESP.sh to do RESP-SigmaHole for your compound
